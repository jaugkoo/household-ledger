# 🚀 빠른 사용법

## 신규 사용자 (처음 설치)

### 1. 설치
```
install.bat 더블클릭
```

### 2. 설정 입력
- OpenAI API Key
- Notion Token
- Notion Database ID
- 모니터링 폴더

### 3. 완료!
컴퓨터 켤 때마다 자동 실행됩니다.

---

## 기존 사용자

### 프로그램 시작
```
start.bat 더블클릭
```

### 설정 변경
```
setup_wizard.py 더블클릭
```

### 자동 시작 제거
```
uninstall.bat 더블클릭
```

---

## 영수증 처리 방법

1. 📱 핸드폰으로 영수증 촬영
2. ☁️ OneDrive 자동 동기화
3. ✨ 자동으로 Notion에 입력됨
